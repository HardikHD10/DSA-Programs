## Setup and Usage

To run any of the C programs:

1. Clone the repository.
2. Navigate to the desired directory or file.
3. Compile the C program:
   ```
   gcc program_name.c -o output_name
   ```
4. Run the compiled program:
   ```
   ./output_name
   ```
